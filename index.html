Virtual Reality DJ (WIP)

Rhythm, Social, VR

I made this for a virtual reality class. Last minute requirements were added to the project that made me make this into an Osu! redesign. However, the original plan for this project was to simulate timecoded concert visuals. I wanted to push the envelope of those visuals to things that can only be currently done in a video game in order to create a unique concert experience. As new 4D immersion technology develops in the real world, this game may be a preview of what that technology can be used for. I am currently working to fulfill the original plans I had for this project.
